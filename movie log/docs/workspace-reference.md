# Workspace Reference

This document defines the approved Movie Log workspace shell for renderer work.

## Required Markers

- ember crown: one dominant warm mass above and behind the working surface
- figure body: one pale lower-center silhouette that carries the primary interaction and reads as the figure, not as a card
- archive band: one dark strap-like archive surface embedded into the figure body instead of detached beside it
- figure halo: one pale circular light source behind the figure body that supports the crown mass
- route talisman: one compact route accessory that reads as part of the poster, not as a sidebar
- blade field: one row of blade-like vertical anchors across the lower edge
- editorial spine: one right-edge vertical label that frames the stage
- warm accent family: one orange-red-cream accent system with black as the field color

## Disallowed Markers

- figure-sheet
- archive-shard
- route-rig
- battle-layout
- route-stack
